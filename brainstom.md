# Project 

Goal of the project is to minimize food waste. 
The means of reaching that goal is an app that helps you manage food inventory. 
- Provide insight in the inventorhy and wastebehaviour. 
- Feedback to user about wastebehaviour.
- Meal recommendations
- Shopping list based on previous behaviour 


# Overall design notes and core concepts


## core concepts
- Expiry date is an extimate. Gradient colour to show possible expiration. Warning look, smell taste, as estimate expiery date appraoches.
- Waste behaviour, metrics about wasted products. 
 

## Designnotes: 
Modern and recent design concepts. 
The Ui needs to be clean and uncluttred. 
Normalized UI
Darkmode



# Page descriptions

## overview screen 
- Inventory summary (estimated value, expired value, 30 day waste.)
- Expiring que with first items to expire (quick action add to shoppinglist). 
- Low stock (quick action add to shoppinglist).
- Most wasted items. 
- Mark meals as eaten. 


## Inventory 
- Ordarable based on product category (vegetables, fruits, dairy etc etc), expiery date
- Able to add new bought foodstuffs. User starts typing and a dropdown does suggestions about existing foodstuffs that are configured in foodstuff configuration page. 
- Also able to add new foodstuff as a quick action from this dropdown as last options in suggestions list. 
- Based on planned meals add flair as planned. (clickable go to meals/recipe)
- Able to select date and grey out foodstuffs in fridge. 

## Meals
Meal recommendations based on whats in the inventory. Prioritize foods that are about to expire. 

- Portion size 
- filterable on allergies
- Favourite recipes. 
- User creatable recipes. 
- Groupable based breakfast, lunch, dinner. 
- Groupable on kitchen of origin (country).
- Plannable meal calender.


## Shoppinglist
Shoppinglist
- Mark meals as eaten. 
- suggested items to put on shopping list( low stock, planned meals)
- Suggestions to improve buying behaviour based on previous expiration and waste behaviour.

A per foodstuff configurable expiry date.
